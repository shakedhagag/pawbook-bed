Technologies

The tecnologies I used in this project are:
Express JS for backend development along with -

  • bcrypt
  • body-parser
  • cookie-parser
  • cors
  • dotenv
  • express
  • express-fileupload
  • express-session
  • jsonwebtoken
  • mongoose
  • multer
NextJS for frontend development along with -
  • axios
  • tailwind CSS
  • react obviously
  • typescript

Challenges:

The challenging part was adapting to the NextJS framework, which was a very fun learning experience! Additionally, setting up the express server was tough since I don't have a background in backend development.
